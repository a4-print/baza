📘 Инструкция по работе с A4-Принт

1. Вход в систему
Введите свой логин и пароль → «Войти».
2. Открытие смены
Нажмите «🚀 Открыть смену» → при необходимости добавьте комментарий (нехватка сдачи, проблемы с техникой и т.д.).
3. Приём заказа
Выберите услугу → введите количество → «Добавить». При необходимости используйте «➕ Быстрая услуга».
4. Оформление заказа
Укажите имя/телефон клиента (опционально), примените скидку, выберите оплату (нал/карта), добавьте комментарий → «Оформить».
5. Печать бланка
Нажмите «🖨️ Бланк заказа» — сформируется А4 с двумя экземплярами (основной + копия клиента).
6. Закрытие смены
В конце дня нажмите «🔒 Закрыть смену» → добавьте итоговый комментарий. Затем «📄 Отчёт» — сохраните PDF для бухгалтерии.

3. Управление операторами
Там же → «Операторы»: смена пароля (🔑), удаление, добавление нового сотрудника.
4. Статистика операторов
Кнопка «📊 Статистика операторов» — заказы, выручка, средний чек по каждому. Можно экспортировать в Excel.
5. Комментарии к смене
При открытии и закрытии смены можно оставлять служебные заметки (финансовые нюансы, поломки, пересменка). Комментарии сохраняются и видны в PDF-отчёте.
